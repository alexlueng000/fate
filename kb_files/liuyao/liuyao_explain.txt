# 六爻解卦方法论：看懂人性的七步法

> **核心信念**：卦不是判官，是镜子。它照的不是吉凶判决，而是人性中的**犹豫、恐惧、欲望与可能**。
> 解卦的终点不是"会不会成"，而是"她为什么这样、我该怎么做、什么时候会变"。

---

## 〇 总纲：三个底层观念

### 1. 卦是看人性，不是看判决
- 没有"好卦坏卦"，只有"此刻的状态"和"还能怎么走"。
- 同一个涣卦，可以解成"她要逃"，也可以解成"你该给空间"——区别在你看的是结果还是过程。
- **解卦的最高目标**：把卦象翻译成对方此刻的"心理动作"，再回推自己该做什么。

### 2. 生克制化是骨，卦名意象是皮
- 卦辞、卦名（"涣""家人""明夷"）只能作辅助联想，**不能用来定吉凶**。
- 真正定调的是：用神旺衰、月日生克、动爻去向。
- 先算账（五行生克），再讲故事（人性翻译）。顺序反了，就是占星。

### 3. 用神看人，世应看关系，动爻看变化
这是分层的，不能混：

| 维度 | 看什么 | 回答什么问题 |
|---|---|---|
| **用神** | 对方这个人本身 | 她现在是什么状态？旺衰空破？ |
| **世应** | 你和她的姿态 | 你们的距离、谁主动、谁回避？ |
| **动爻** | 事情的发动机 | 接下来会怎么变？变向哪里？ |
| **变卦** | 事情的归宿 | 最终落到什么状态？ |

---

## 一 起卦：一念一占

- **一卦只问一事**。心怀两三事而占者，必不灵验。
- **自占感情、功名宜独占**，事过再问人。代占常常因为"心思不专一"而读不准。
- **节节问去**：先问眼下，再问近期，再问长远。神报近事多于远事，一卦想包打终身就糊。
- **卦象不明时，再占一卦合参**——这是野鹤老人的标准做法，比硬解一卦稳得多。

---

## 二 装卦：建立卦体（顺序不能错）

| 步骤 | 内容 |
|---|---|
| ① | 摇出本卦六爻、记录动爻位置 |
| ② | 由本卦定宫，按"生我父母、我生子孙、我克妻财、克我官鬼、比和兄弟"装六亲 |
| ③ | 按浑天甲子装地支五行 |
| ④ | 标世应（隔两位为应） |
| ⑤ | 动爻变出之爻——**六亲仍照本卦之宫所属来定**（关键易错点）|
| ⑥ | 按日干起六神（青龙、朱雀、勾陈、螣蛇、白虎、玄武，自下而上） |
| ⑦ | 标记日辰、月建、旬空 |

---

## 三 第一步：定用神（解卦的真起点）

### 用神归类速查

| 所占之事 | 用神 |
|---|---|
| 男占女（感情、女方状态） | **妻财爻** |
| 女占男（感情、男方状态） | **官鬼爻** |
| 占父母、长辈、师长、文书、信息、屋宅、车物 | **父母爻** |
| 占功名、官职、雷霆、贼盗邪祟、疾病本身 | **官鬼爻** |
| 占兄弟、同辈、朋友、谋事阻隔者、劫财之人 | **兄弟爻** |
| 占妻妾、奴仆、钱财、所驱使之人 | **妻财爻** |
| 占子女、晚辈、医药、僧道、解忧之神 | **子孙爻** |
| 自占吉凶、自身之事 | **世爻** |
| 占他人、对手 | **应爻** |

### 四神定位

- **用神**：所问之事的核心爻。
- **元神**：生用神之爻（援助力量）。
- **忌神**：克用神之爻（伤害力量）。
- **仇神**：克元神、生忌神之爻（间接为害）。

> 例：金为用神 → 土为元神，火为忌神，木为仇神。

### 用神不上卦怎么办
1. 先**多占**一卦，三占仍不现 → 取**伏神**。
2. 若月建即是用神之五行 → **直接以月建为用神**。

---

## 四 第二步：四方查旺衰（算清这笔账）

对**用神、元神、忌神**逐一做四方扫描：

| 维度 | 问什么 |
|---|---|
| **月建** | 当月生它、克它、临它、冲它（月破）？ |
| **日辰** | 当日生它、克它、临它、冲它？ |
| **动爻** | 卦中动爻里有没有生它或克它的？ |
| **变爻 / 回头** | 自己动者，化出何爻？回头生还是回头克？ |

### 旺衰判断的关键术语

| 状态 | 含义 |
|---|---|
| **旺相** | 临月建、日辰，或得生扶 |
| **休囚** | 月日不助，无生扶 |
| **旬空** | 旬内为空——动空、旺空待时而用；衰空真空到底为空 |
| **月破** | 月建冲之而无救——出月、填实、合日方应 |
| **暗动** | 旺静爻被日辰冲，悄然发动，吉凶照应 |
| **日破** | 衰静爻被日辰冲，愈加无用 |
| **进神** | 寅化卯、巳化午、申化酉、亥化子——势进，吉凶皆增 |
| **退神** | 卯化寅、午化巳、酉化申、子化亥——势退，吉凶皆减 |
| **回头生** | 化出之爻反生本爻——援助 |
| **回头克** | 化出之爻反克本爻——纵旺亦凶 |
| **入墓** | 入日墓、动墓、化墓——滞而无用，待冲开方应 |

### 元神有力的五个标志
1. 元神旺相、临日月或得日月动爻生扶。
2. 元神动而化回头生、化进神。
3. 元神长生、帝旺于日辰。
4. 元神与忌神同动（贪生忘克，化险为夷）。
5. 元神旺动而临空、化空（待出空仍可用）。

### 忌神无力的七个标志
1. 忌神休囚不动，或动而被克。
2. 忌神静而临空破。
3. 忌神入三墓。
4. 忌神衰而化退神。
5. 忌神衰而又绝。
6. 忌神动化绝、克、破、散。
7. 忌神与元神同动。

---

## 五 第三步：看世应关系（你和她的姿态）

世为己，应为人。这一步看的不是"她是谁"，而是"你俩在此刻的相对位置"。

### 关键观察点

- **应爻临何六亲**——揭示她的姿态：
  - 临财：她对你有意（男占）
  - 临鬼：她有压力，或心里有别人
  - 临兄：有第三者、竞争、隔阻
  - 临子：她想自由、想躲、不想承担
  - 临父：她在思虑、有顾虑、被规则约束

- **世应生克**：
  - 应生世 = 她向你来
  - 世生应 = 你在付出
  - 应克世 = 她对你有压力或敌意
  - 世克应 = 你在主导
  - 世应相合 = 心意相通
  - 世应相冲 = 此刻对峙

- **世应俱空** = 双方都没准实，谋事荒唐。
- **世空** = 你自己心不定；**应空** = 她不实。

---

## 六 第四步：看动爻（事情的发动机）

> 动爻是钥匙。一卦的故事，主要由动爻讲完。

### 动爻读法

| 情况 | 解法 |
|---|---|
| **独发**（只一爻动）| 它就是断卦的核心，事情围绕它展开 |
| **多爻动** | 分清谁是元神动、谁是忌神动；忌神动而元神同动 = 贪生忘克 |
| **化进神** | 势头加强，吉则更吉、凶则更凶 |
| **化退神** | 势头减弱，吉减、凶也减 |
| **化回头生** | 援助到位 |
| **化回头克** | 纵旺亦凶——这是判断转折的硬信号 |
| **化空 / 化破** | 暂时无果，待出空、填实方应 |
| **化合 / 化绊** | 被绊住了，待冲开之日方动 |

### 卦体变化的整体信号

| 卦变 | 含义 |
|---|---|
| **六冲变六合** | 先散后聚，散而复合（感情上很好的信号）|
| **六合变六冲** | 先合后散，始吉终凶 |
| **六冲变六冲** | 始终冲散，难成 |
| **六合变六合** | 始终和合，长久 |

---

## 七 第五步：读人事物（把卦翻译成人性）

这是最有灵性的一步——**把五行账目翻译成具体的现实动作**。

### 翻译模板

| 卦象 | 现实翻译 |
|---|---|
| 父母爻临日辰、生世 | 你的表达（信息、文字、作品）此刻很有力量 |
| 父母爻克应爻 | 你的心意正在触达她，但她还没接住 |
| 应爻伏吟 / 反吟 | 她反反复复、心绪不定 |
| 应爻在初爻临子孙 | 她姿态低、想躲、想轻松 |
| 妻财爻空 | 她此刻不在状态、没回应能力 |
| 妻财爻动而化回头生 | 她内心有变化的种子 |
| 兄弟爻动 | 有第三者、有口舌、有破财之兆 |
| 子孙爻持世 | 你想要自由、不想承担 |
| 官鬼爻动而克世 | 有压力、有麻烦、对方让你不安 |

### 翻译的两个原则

1. **看过程和状态，不只看结果**——卦告诉你"她在害怕"，比告诉你"会成"更有用。
2. **吉凶里有方位、有人物、有动作**——同一个"凶"字，可以是争吵，可以是冷淡，可以是第三者，分清楚才有用。

---

## 八 第六步：判结果（综合大象）

### 综合判断的优先级

1. **用神是否旺相** —— 没这个底子，一切免谈
2. **元神是否有力** —— 决定能不能撑住
3. **忌神是否无力** —— 决定有没有暗坑
4. **大象是否生合世（用）** —— 决定方向是否相向
5. **本卦→变卦的卦体变化** —— 决定终局走向

### 几条判断捷法

- **用神旺 + 元神有力 + 忌神无力 + 世应相生** = 大吉
- **用神空破 + 忌神当令 + 元神受伤 + 世应相冲** = 大凶
- **六冲变六合** + 用神有根 = 即使现在散，最终能合
- **六合变六冲** + 用神受克 = 即使现在好，难有终
- **回头克的动爻** = 看着热闹，最终翻车

---

## 九 第七步：推应期（这一步最容易错）

> 应期判断错，整盘解卦就全错了方向。

### 应期总诀

| 用神状态 | 应在 |
|---|---|
| **静而旺** | 逢值日（用神地支当日）或逢冲日 |
| **动而旺** | 逢合日 或 逢值日 |
| **太旺** | 逢墓日 或 逢冲日 |
| **衰绝** | 逢生扶之月日，或当令之季节 |
| **旬空** | 出空之日、冲空之日、填实之日 |
| **月破** | 出月、填实日、合日；过月不破 |
| **入墓** | 冲开墓库之日 |
| **逢合（被绊住）** | 冲开合的那一天 |
| **化进神** | 逢值、逢合之日 |
| **化退神** | 逢值、逢冲之日 |

### 应期的层次

- **远事**：定之于年、月。
- **近事**：应之于日、时。
- **占月应年、占日应时**——须随事斟酌。

### 应期判断的关键陷阱

> **应期判断前，必须先判用神旺衰。**
>
> - 旺者被冲 = **冲起 / 暗动**（确实会主动来）
> - 衰者被冲 = **日破**（不仅不会来，反而更躲）
>
> 同样一个"日辰冲应爻"，旺衰不同，应法相反。

---

## 十 心法总结：解卦六问

每解一卦，按这六问走一遍，就不会漏：

1. **谁是用神？** —— 这一卦在问什么人、什么事？
2. **她（用神）现在什么状态？** —— 旺衰空破，被谁生被谁克？
3. **你和她（世应）什么姿态？** —— 谁主动谁回避？相生还是相克？
4. **谁在动（动爻）？要变向哪里？** —— 找到事情的发动机和方向。
5. **大象是什么走势（变卦）？** —— 散而复聚，还是合而终散？
6. **什么时候会变（应期）？** —— 先判旺衰，再套应期总诀。

---

## 十一 给"看懂人性"的几条额外提醒

1. **卦不是来告诉你结果的，是来告诉你局势的。** 你要的不是"会成吗"，是"她在哪个心理位置、我该往哪个方向用力"。

2. **没有坏卦，只有不会读的卦。** 涣卦可以解成"她要逃"，也可以解成"你不能再紧逼，要给空间"。前者让你绝望，后者让你行动——选哪个，看你想从卦里要什么。

3. **学解卦，本质是在学看懂人性中的犹豫与恐惧。** 五行生克是工具，最终落点是人。一个解卦者最珍贵的能力，不是算得准，是**看得懂对方为什么这样**。

4. **多占合参，胜过死断一卦。** 疑则再占，不为渎神。野鹤老人毕生精验，全在"多占合参"四个字。

5. **解卦的终点是行动，不是预测。** 卦给你的不是命运的剧本，是此刻的镜子——照清楚了，怎么走是你自己的事。

---

> **一句话收束**：
> 先算清五行的账，再读懂人性的局；
> 用神看她，世应看你俩，动爻看变化，应期看时辰；
> 卦不判命，卦只照人。
